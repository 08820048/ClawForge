# Forclaw

Forclaw is a desktop application built with Tauri + React. This repository contains the app source code and build tooling.

## Features

- Fast desktop shell with Tauri
- Modern React + Vite frontend

## Tech Stack

- Tauri v2
- React 19
- Vite
- TypeScript

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- Rust toolchain
- Tauri prerequisites for your OS

### Install

```bash
npm install
```

### Run (Frontend)

```bash
npm run dev
```

### Run (Desktop App)

```bash
npm run tauri dev
```

### Build

```bash
npm run build
npm run tauri build
```

## Project Structure

- `src/` React UI
- `src-tauri/` Tauri backend and bundle configuration
