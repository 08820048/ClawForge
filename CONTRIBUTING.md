# Contributing

Thanks for your interest in contributing to ClawForge.

## Quick Start

1. Fork the repo and create a feature branch.
2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm run tauri dev
```

4. Before opening a PR, ensure builds pass:

```bash
npm run build
npm run tauri build
```

## Guidelines

- Keep changes focused and scoped.
- Prefer small PRs with clear descriptions.
- Update documentation when behavior changes.
- There are no automated tests yet. If you add tests, include how to run them in the PR description.

## Commit Messages

Use concise, imperative messages (e.g. "Add settings panel").
